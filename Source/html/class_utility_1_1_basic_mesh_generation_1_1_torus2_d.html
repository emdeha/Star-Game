Utility::BasicMeshGeneration::Torus2D::Init ( )

Generates the vertex and index data for the object and binds OpenGL's buffers.

For the two radiuses we use a formula similar to the one calculating the satellite's position (see Satellite::Update()). That way we generate the vertex data.

The index data is little more complex. We will be rendering via triangle strips. So, we need the first index to be the first outer radius' vertex and the second index to be the first inner radius' vertex. That is repeated for all vertices. At the end we 'sew' the torus (see the three lines of code after the for cycle).

The buffers initialization is straightforward if you are familiar with OpenGL. For further research see this book.
