\subsection{Detailed Description}
Represents a sun's satellite. 

A satellite has a parent around which it rotates. It also has a diameter. A satellite checks for clicks on it, loads on-\/click animations, etc.

Checks for click on the satellite. 

In order to check for click we need to do two things\-:
\begin{DoxyItemize}
\item To get the mouse ray from the camera to the object
\item To check for intersections between the ray and the object Getting the ray is achieved with the user\-Mouse.\-Get\-Pick\-Ray() function.
\end{DoxyItemize}

The test against collision of the ray with an ellipsoid because the camera is at an 135.\-0 degree angle relative to the scene's plane.
