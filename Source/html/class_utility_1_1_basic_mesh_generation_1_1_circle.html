Utility::BasicMeshGeneration::Circle::Init ( )

Generates the vertex and index data for the object and binds OpenGL's buffers.

The generation of the vertex data is pretty straightforward (see Satellite::Update()).

The index data is simply every vertex of the circle.

The buffers initialization is straightforward if you are familiar with OpenGL. For further research see this book.
